Sobre o Projeto

O CoachAI foi desenvolvido com o objetivo de auxiliar utilizadores durante exercícios físicos, analisando a postura corporal através da câmara do dispositivo. A aplicação identifica pontos-chave do corpo humano e apresenta feedback automático sobre enquadramento, iluminação, posicionamento e visibilidade corporal.

Tecnologias Utilizadas

Kotlin Android Studio CameraX MediaPipe Pose Landmarker XML Layouts GitHub

Funcionalidades Principais

Deteção Corporal

Identificação de 33 pontos-chave do corpo humano através do MediaPipe Pose Landmarker.

Feedback em Tempo Real

Mensagens automáticas para melhorar postura, distância, iluminação e enquadramento.

Processamento de Vídeo

Análise de vídeos de exercícios para testar o desempenho dos modelos de pose estimation.

Overlay Visual

Desenho do esqueleto corporal sobre a imagem capturada pela câmara.

Metodologia

1. Captura

A imagem é obtida através da câmara ou de um vídeo carregado.

2. Deteção

O modelo identifica os pontos corporais relevantes.

3. Análise

São avaliadas condições como iluminação, visibilidade e posicionamento.

4. Feedback

O utilizador recebe indicações para corrigir a execução.

Resultados Experimentais

Foram realizados testes comparativos entre diferentes modelos de Human Pose Estimation. O MediaPipe Pose Landmarker apresentou melhor desempenho para utilização mobile em tempo real. O YOLO Pose foi também testado, mas obteve menor desempenho médio. RTMPose e ViTPose foram analisados, embora não tenha sido possível concluir os testes devido a incompatibilidades de dependências.
