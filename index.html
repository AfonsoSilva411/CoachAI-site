CoachAI

Aplicação Android para análise postural e feedback em tempo real usando Human Pose Estimation.

Ver Projeto

Sobre o Projeto

O CoachAI é uma aplicação Android desenvolvida em Kotlin que utiliza o MediaPipe Pose Landmarker para detetar pontos-chave do corpo humano e fornecer feedback automático durante exercícios físicos.

Objetivos

Análise postural automática
Feedback em tempo real
Deteção de keypoints corporais
Apoio à prática de exercício físico

Funcionalidades

Deteção de 33 pontos corporais
Processamento de vídeo
Câmara em tempo real
Análise de iluminação
Avaliação do enquadramento
Overlay do esqueleto

Tecnologias Utilizadas

Kotlin Android Studio CameraX MediaPipe Pose Landmarker XML Layouts

Arquitetura

CameraX → MediaPipe Pose Landmarker → Análise Postural → Feedback ao Utilizador

Resultados

Foram realizados testes com diferentes modelos de Human Pose Estimation. O MediaPipe Pose Landmarker apresentou melhor desempenho para uso mobile em tempo real, enquanto o YOLO Pose teve desempenho inferior. RTMPose e ViTPose foram analisados, mas não foi possível concluir os testes devido a problemas de dependências.
